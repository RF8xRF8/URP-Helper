# URP 抢课助手

一个面向 URP 系统的可视化选课工具。

## 快速开始

1. 先在 [app.py](app.py) 中填写你学校的 URP 地址配置（`URP_CONFIG`）。
2. 双击 [start.bat](start.bat) 启动。
3. 浏览器打开 `http://127.0.0.1:5000`。

## 页面怎么用

1. 填写账号和密码。
2. 选择登录方式：
- 校园网内：直连
- 校园网外：WebVPN(只适配了本人所在学校)
3. 添加要抢的课程。
4. 点击“开始抢课”。
5. 过程中如出现多条匹配，按弹窗选择目标课程。

## 课程填写建议

- 推荐优先填写 `kch + kxh`，成功率更高。
- 只填课程名或教师名时，匹配结果可能较多。

## 常见提示

### OCR runtime check failed

请安装 Microsoft Visual C++ Redistributable 2015-2022 x64：

`https://aka.ms/vs/17/release/vc_redist.x64.exe`

安装后重新启动 [start.bat](start.bat)。

### 登录失败

- 先确认账号密码是否正确。
- 校外网络请切换到 WebVPN 登录。

### 没有查到课程

- 检查课程号、课序号是否填写正确。
- 确认当前是否处于可选课时间。

## 使用须知

- 请仅用于学习和个人使用。
- 请遵守学校相关规定与法律法规。
